_**Assistant**_

```css:style.css
.overlay-title {
  // ... bestehender Code ...
}

/* Hinzufügen der overlay-character Klasse unter dem overlay-title */
.overlay-character {
  width: 350px; /* Breite des Charakters anpassen */
```
